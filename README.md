# vibe-ab-min
最小版 A/B 文案產生器（純前端）。開啟 `index.html` 即可。

## 使用
1. 開啟 `index.html`。
2. 輸入主題與賣點 → 產生 A/B 兩版。
3. 點「切換語氣」看看不同 vibe（青春直白 / 理工理性 / 溫暖關懷）。

## 結構
- index.html — UI
- app.js — 生成邏輯（無框架）
- styles.css — 簡單樣式
